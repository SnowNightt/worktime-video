## 正在开发的模块

### 音乐模块

在 VS Code 侧边栏 Webview 中提供一个听歌入口。

- 完善歌单详情、歌曲列表和播放状态展示。
- 增加更完整的播放控制，例如切歌、播放模式、进度展示等。
- 优化异常提示、加载状态和登录失效处理。

## 未来计划：Bilibili 模块

Bilibili 模块是后续打算开发的摸鱼模块，目标是在 VS Code 侧边栏里提供视频观看。

计划开发方向包括：

- B 站内容入口：展示推荐视频、热门内容或指定分区内容。
- 视频搜索：支持按关键词搜索视频。
- 视频播放：在 Webview 内播放视频。
- 收藏与稍后再看：提供常用视频的快捷访问入口。
